Blog Console

Manage your articles by requesting the following API endpoints:

Users

  • Create an account with POST users/sign-up
  • Get your JSON Web Token (JWT) authorization with POST users/sign-in

Authorization tokens lasts up to three months.

Articles

  • Get all articles with GET articles. It accepts the following options:
    • Filter by privacy: GET articles?isPublic=[true|false]
    • Filter by user: GET articles?userId=[number]
    • Find a specific article: GET articles/[id]
  • Create an article with POST articles
  • Edit an article with PATCH articles/[id]
  • Delete an article with DELETE articles/[id]

Comments

  • Get all article comments with GET articles/[id]/comments
  • Create a comment with POST articles/[id]/comments
  • Edit a comment with PATCH articles/[id]/comments/[id]
  • Delete a comment with DELETE articles/[id]/comments/[id]
